
# 📰 News Headlines Web Scraper

A Python CLI application that scrapes the latest news headlines using **requests** and **BeautifulSoup**, displays them in the console, and saves them to `headlines.txt`.

## Files

- `news_scraper.py`
- `headlines.txt`
- `requirements.txt`
- `README.md`

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python news_scraper.py
```
